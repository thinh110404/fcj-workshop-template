### Week 5 Objectives:

- Review the knowledge of Amazon EC2, EBS, Security Group, and basic networking.
- Practice building and connecting to an EC2 server.
- Understand the basic process of setting up a VPN connection.
- Work directly at the company office and participate in group discussions.
- Improve troubleshooting skills when configuring EC2 and networking.

### Week 5 Achievements:

- Reviewed and strengthened knowledge of Amazon EC2 and its main components:
  - EC2 instance
  - AMI
  - Instance type
  - Key pair
  - Security Group
  - EBS volume

- Practiced launching and configuring an EC2 instance.

- Understood how inbound rules in the Security Group affect the ability to connect to an EC2 instance.

- Participated in group discussions and practical work related to building an EC2 server and setting up VPN connectivity.

- Learned several troubleshooting steps for common EC2 connection issues, including:
  - Checking the key pair
  - Checking the public IP address
  - Checking the inbound rules of the Security Group
  - Checking network and route configuration
  - Checking the instance status

- Understood the basic purpose of VPN in securely connecting private networks.

- Gained practical experience working with AWS resources and resolving configuration issues during lab exercises.
